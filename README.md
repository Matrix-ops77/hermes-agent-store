## Quick Start

```bash
# Clone
git clone https://github.com/Matrix-ops77/hermes-agent-store.git
cd hermes-agent-store

# Install
npm install

# Dev
npm run dev

# Build
npm run build

# Deploy (requires Cloudflare API token)
npm run deploy
```

## Stack

- **Next.js 15** (App Router, RSC)
- **TypeScript**
- **Tailwind CSS v4**
- **Cloudflare Pages** (edge deployment)
- **GitHub Actions** (CI/CD)

## Deployment

1. Fork this repo
2. Add `CLOUDFLARE_API_TOKEN` and `CLOUDFLARE_ACCOUNT_ID` to GitHub Secrets
3. Push to `main` — auto-deploys to Cloudflare Pages
